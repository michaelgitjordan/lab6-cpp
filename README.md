# Класс 1: TouristSet (Неупорядоченное множество)

## Назначение
1. Класс реализует функционал неупорядоченного множества, которое:

2. Содержит только уникальные элементы (дубликаты игнорируются)

3. Не предоставляет доступ к элементам по индексу

4. Основан на массиве (std::vector) для хранения элементов

## Основные возможности

1. Добавление элементов: add() - добавляет элемент, если его еще нет в множестве

2. Удаление элементов: remove() - удаляет указанный элемент

3. Проверка наличия: contains() - возвращает true/false

## Операции над множествами:

1. Union() - объединение двух множеств

2. Except() - разность множеств (элементы из первого, которых нет во втором)

3. Intersect() - пересечение множеств (общие элементы)

# Класс 2: StudentDictionary (Словарь на основе хэш-таблицы)

## Назначение
Пользовательская реализация словаря (ассоциативного массива) с поддержкой уникальных ключей. Реализована без использования стандартных контейнеров STL.

## Архитектура
- Хэш-таблица с фиксированным размером 101 (простое число)

- Метод цепочек для разрешения коллизий

- Хэш-функция: полиномиальный хэш для строк

## Основные операции
1. Вставка: insert() - добавляет пару ключ-значение (ключи уникальны)

2. Удаление: remove() - удаляет элемент по ключу

3. Поиск: get() и contains() - поиск по ключу

4. Очистка: clear() - полная очистка словаря

5. Загрузка из файла: loadFromFile() - загрузка данных из текстового файла

# Класс Automatic (Автомат)

## Описание:

Класс Automatic представляет собой автоматическое оружие, которое является расширенной версией обычного пистолета. Основное отличие — поддержка скорострельности (fire rate), что позволяет производить несколько выстрелов за одну команду.

# Основные возможности:

## Конструкторы:
1. Automatic() — создает автомат с 30 патронами и скорострельностью 30 выстрелов/сек

2. Automatic(int maxBullets) — создает автомат с указанным количеством патронов, скорострельность = половина обоймы

3. Automatic(int maxBullets, int fireRate) — создает автомат с указанными патронами и скорострельностью

## Методы:

1. void shoot() — производит выстрелы в соответствии со скорострельностью (например, 3 выстрела при fireRate=3)

2. void shootSeconds(int seconds) — стреляет указанное количество секунд (количество выстрелов = секунды × скорострельность)

3. void reload(int newBullets) — перезаряжает автомат

4. void print() const — выводит информацию об автомате

5. int getBullets() const — возвращает количество оставшихся патронов

6. int getFireRate() const — возвращает скорострельность
