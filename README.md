# Класс Automatic (Автомат)

## Описание:

Класс Automatic представляет собой автоматическое оружие, которое является расширенной версией обычного пистолета. Основное отличие — поддержка скорострельности (fire rate), что позволяет производить несколько выстрелов за одну команду.

# Основные возможности:

## Конструкторы:
1. Automatic() — создает автомат с 30 патронами и скорострельностью 30 выстрелов/сек

2. Automatic(int maxBullets) — создает автомат с указанным количеством патронов, скорострельность = половина обоймы

3. Automatic(int maxBullets, int fireRate) — создает автомат с указанными патронами и скорострельностью

## Методы:

1. void shoot() — производит выстрелы в соответствии со скорострельностью (например, 3 выстрела при fireRate=3)

2. void shootSeconds(int seconds) — стреляет указанное количество секунд (количество выстрелов = секунды × скорострельность)

3. void reload(int newBullets) — перезаряжает автомат

4. void print() const — выводит информацию об автомате

5. int getBullets() const — возвращает количество оставшихся патронов

6. int getFireRate() const — возвращает скорострельность
